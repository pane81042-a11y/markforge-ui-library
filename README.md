# MarkForge UI Library

MarkForge UI Library is a shared component system used across all MarkForge microfrontends.

It ensures consistent design, reusable components, and maintainable frontend architecture.

## Purpose

- Maintain visual consistency
- Reduce duplicated UI code
- Centralize design updates
- Improve scalability across microfrontends

## Components

- Buttons
- Forms and Inputs
- Modals
- Layout components
- Typography system
- Notification components
- Loading indicators
- Error states

## Architecture

Built with:

- React
- TypeScript (recommended)
- CSS Modules or Styled Components
- Shared design tokens

Used as a shared dependency in:

- markforge-shell-frontend
- markforge-editor-frontend
- markforge-version-frontend
- markforge-export-frontend
- markforge-auth-frontend
- markforge-admin-frontend

## Distribution

- Private npm package or GitHub Package Registry
- Version controlled releases
- Integrated via package manager

## Portfolio Value

Demonstrates:

- Design system architecture
- Reusable component engineering
- Scalable frontend structure
- Enterprise UI standardization
